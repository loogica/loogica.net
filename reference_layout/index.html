Nós temos como objetivo promover a criação de software Open Source no Brasil

Queremos criar e desenvolver uma cultura de colaboração e pesquisa pra que a nossa criatividade se some a técnica levando a criação de soluções livres que poderão ajudar outros profissionais a criar soluções para problemas do "mundo real".

A nossa idéia é atuar com estudantes, desenvolvedores e com outros grupos ou empresas que promovem a criação de tecnologia nacional.

A Loogica é a soma do software open source com a cultura de colaboração.
